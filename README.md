# Segment koda sa složenim komponentama

Primjer ilustrira primjenu children svojstva u React komponenti.

Dodatno, ilustrira se napredniji obrazac dizajna React komponenti zvan složene komponente (_eng. compound components_).

Komponenta koja služi kao semafor rezultata nogometnog kluba na nekoj utakmici.
